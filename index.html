Setting up the Repo

  • Install Sass
  • Build Sass
    • In terminal at project root
      • One compile: sass .
      • Watch for changes: sass --watch .
    • Or use as extension like Live Sass Compiler

Using Jekyll

  • Install Ruby, Jekyll, and Bundler
  • Update _config.yml - not necessary for local running
  • In a terminal run:
    • bundle install
    • bundle exec jekyll serve
    • Alternatively, use a vs code extension like Jekyll Run
  • Preview site at http://localhost:4000/css-framework/

Without Jekyll

  • Use an extension like Live Server to run the application
  • Navigate to /no-build and run from there
  • make any html changes to the files in no-build instead of the top level files
